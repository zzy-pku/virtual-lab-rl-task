# RLinf 任务地图

本文件回答三个问题：

1. 当前任务线在做什么；
2. 四个等级到底训练什么能力；
3. 为什么最后会走到 RLinf + Genesis 的 Level 3。

---

## 1. 当前任务线的主问题

当前虚拟科研实习室只开放一条主线任务：

> **基于 RLinf 的 π0.5 强化学习训练任务族**

其训练对象是 vision-language-action 模型 π0.5；
其训练框架是 RLinf；
其起始环境是 RLinf 官方已支持的标准 benchmark；
其高难度目标是将这一训练链路扩展到新的 Genesis 仿真环境中。

这一任务线在研究上主要落在：

- 具身智能
- 机器人操作
- 强化学习 fine-tuning
- VLA 模型在机器人任务中的应用
- 仿真环境与训练框架整合

---

## 2. 这条任务线的统一训练目标

你在这条任务线中最终要学会的，不是某一条命令，而是完整理解下面这条链路：

**环境观测 -> 语言任务描述 -> π0.5 产生动作 -> RL 更新策略 -> 评测结果 -> 分析与改进**

所以这条任务线的训练目标可以拆成五项：

1. 理解 RLinf 项目结构和训练入口；
2. 理解 π0.5 模型在任务中的输入输出形式；
3. 理解 benchmark / simulator / observation / action 这些对象如何连接；
4. 学会组织训练、评测和结果分析；
5. 在高难度阶段学会将成熟 RL 框架扩展到新环境中。

---

## 3. 三个等级的能力递进

### Level 1：完整项目运行与结果观察
你得到的是：简化的训练代码、标准训练命令、现成配置与可运行项目。

你要训练的是：
- 会跑通训练；
- 会读日志；
- 会改少量参数；
- 会做最基本的结果比较。


### Level 2：标准环境内的自主任务组织与实验搭建
你得到的是：框架边界和 benchmark 边界。

你要训练的是：
- 在 RLinf 已支持的标准环境内自己选一个具体任务方向；
- 自己组织配置、训练入口和评测流程；
- 自己做至少一组结果分析。

### Level 3：RLinf 与 Genesis 仿真环境整合
你面对的是：新仿真环境接入问题。

你要训练的是：
- 理解 RLinf 的环境抽象与接口边界；
- 把 Genesis 环境接入训练链路；
- 完成新环境中的训练与验证。


---

## 5. 推荐进入方式

- 想先建立整体直觉：从 Level 1 开始；level1代码就在本仓库内
- 已经能跑标准例子，想训练独立组织实验：进入 Level 2；level2不提供代码，自行按照官方文档进行，
- 已经掌握前两级并且与你的课题相关：再进入 Level 3。level3示例代码在于：https://gitee.com/xujiadong/rlinf-digital-twin(尚未开源)

---

## 6. 你现在最应该记住的一句话

> 这条任务线的核心，不是“把 RLinf 跑起来”，而是通过 RLinf + π0.5 这条真实项目线，逐步获得运行、接入、组织实验和扩展框架的能力。
